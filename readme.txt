Send email to post commenters

== Description ==

Send email to your post commenters with 2 clicks. If you want to send email to users who commented on a certain post,
you can do that easily. Also you can select all the commenters from your site to send email.

= Github =
Visit the [Github](https://github.com/tareq1988/commenter-email) for the latest development snapshot.

== Installation ==

1. Upload and install the plugin
1. You'll have a new options page under the 'Settings' area

== Frequently Asked Questions ==

= Can I select other post types? =

No, this plugin only support `post` post type.

== Screenshots ==

1. You can select a single post or all posts.
2. User email from that post or from all post will be prensented there as Bcc. Now you can send the mail
